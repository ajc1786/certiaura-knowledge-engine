# Order-to-Product Passport Gate

Every order line must reference an active Product Passport™ and the precise batch evidence used for release. Public display approval, marketplace eligibility, batch verification and legal supply status are independent checks.

Any passport suspension, expiry, supersession, evidence withdrawal, batch mismatch or critical alert blocks new order release for the affected scope. Existing distributed stock follows the incident and recall workflow rather than silent deletion or retroactive status changes.

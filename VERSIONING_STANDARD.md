# Versioning Standard

Certiaura uses semantic-style versioning.

## Format

```text
vMAJOR.MINOR.PATCH
```

## Rules

| Change type | Example | Version impact |
|---|---|---|
| Major structural change | New asset model | v2.0.0 |
| Meaningful content expansion | New evidence section | v1.1.0 |
| Minor correction | Typo or formatting correction | v1.1.1 |

## Asset maturity labels

- Draft
- In Production
- Review
- Certified
- Published
- Superseded

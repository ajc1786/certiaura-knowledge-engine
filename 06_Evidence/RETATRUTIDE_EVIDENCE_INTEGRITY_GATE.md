# RETATRUTIDE EVIDENCE INTEGRITY GATE

Before an Evidence Object can contribute to a future Platinum certification, it must pass the following gates.

## Gate 1 — Source identity

- Journal / registry / sponsor identified
- Persistent identifier recorded where available
- Publication or result date recorded

## Gate 2 — Direct source access

- Primary source directly accessed
- Access status recorded
- Paywall / access block recorded honestly

## Gate 3 — Quantitative extraction

- Population extracted
- Intervention extracted
- Comparator extracted
- Duration extracted
- Endpoints extracted
- Numerical results extracted
- Confidence intervals / uncertainty captured where available

## Gate 4 — Safety extraction

- Adverse events
- Serious adverse events
- Discontinuations
- Dose relationship
- Titration context
- Study-specific limitations

## Gate 5 — Interpretation discipline

- Primary findings separated from interpretation
- Sponsor context recorded
- No cross-trial comparison presented as randomised head-to-head evidence unless it actually is
- No secondary-reported number presented as primary-verified

## Gate 6 — Relationship integrity

- Evidence links to the correct peptide asset
- Condition relationships
- Monitoring relationships
- Safety relationships
- Goal relationships

## Current Retatrutide rule

An unresolved primary-source gap blocks **domain closure**, but does not erase the value of already verified evidence.

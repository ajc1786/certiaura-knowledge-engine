# RETATRUTIDE USER LAYER MAP

The same canonical asset can serve different user layers without duplicating the underlying evidence.

## Layer 1 — Basic / Lay Summary

Show:
- What Retatrutide is
- What receptor systems it targets
- What major trial domains exist
- High-level evidence status
- High-level safety domains
- What remains uncertain

Hide by default:
- Dense statistical detail
- Internal source-closure mechanics

## Layer 2 — Informed Research User

Show:
- Study-by-study evidence
- Dose groups within trial context
- Outcome tables
- Safety and monitoring domains
- Source status
- Evidence gaps

## Layer 3 — Professional / Technical

Show:
- Evidence Object links
- Source-closure state
- Trial design
- Statistical estimands
- Confidence intervals where captured
- Safety tables
- Conflict / sponsor context
- Relationship graph

## Layer 4 — Enterprise / API

Expose:
- UAI identifiers
- Machine-readable JSON
- Relationships
- Evidence status
- Claim controls
- Versioning
- Change history

## Rule

User layers change presentation depth.

They do not change the canonical underlying facts.

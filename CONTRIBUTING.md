# Contributing to Certiaura Knowledge Engine

## Purpose
This repository contains Certiaura's structured scientific knowledge assets and related platform documentation.

## Working rules
- All meaningful work must be committed to GitHub.
- No agreed standard is changed unless explicitly superseded.
- Every asset must use the Universal Asset Identifier format.
- Every asset must include version, status and relationship metadata.
- Markdown is the preferred human-readable source format.
- JSON is the preferred machine-readable format.

## Commit message examples
- Add Retatrutide evidence object template
- Update Master Asset Register
- Expand GLP-1 receptor biology object
- Fix repository validation workflow

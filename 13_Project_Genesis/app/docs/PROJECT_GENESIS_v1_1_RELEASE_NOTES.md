# Project Genesis v1.1.0 Release Notes

## Added

- Functional Build Pack Import.
- Import preview before files are copied.
- New-file and replacement-file counts.
- Repository / Build Pack path safety checks.
- Automatic creation of required destination folders.
- Repository dashboard refresh after successful import.
- Basic Build Pack Import test.

## Still intentionally excluded

- Automatic Git commit.
- Automatic Git push.
- GitHub release creation.
- Automatic Master Asset Register mutation.
- Automatic Change Log mutation.

These remain separate development steps to avoid combining file import and repository publication before the import workflow has been proven.

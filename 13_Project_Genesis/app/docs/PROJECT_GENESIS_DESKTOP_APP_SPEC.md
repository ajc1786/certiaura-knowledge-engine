# Project Genesis Desktop App Specification

## Objective
Create a no-code desktop repository manager for Certiaura.

## v1.0 Scope
- Select local repository folder
- Validate required folder structure
- Validate core documents
- Display basic repository dashboard
- Open Master Asset Register

## v1.1 Scope
- Import Build Pack from extracted folder
- Copy files into correct repository locations
- Update Change Log
- Update Dashboard

## v1.2 Scope
- Git commit and push from the app
- Suggested commit messages
- Build number tracking

## v1.3 Scope
- Asset ID validation
- Relationship validation
- Completeness score calculation

## Long-term Scope
- AI-assisted asset creation
- Evidence object builder
- Release creation
- Knowledge graph export
- Publishing engine integration

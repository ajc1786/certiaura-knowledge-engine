# Project Genesis v1.0 — Desktop App Scaffold

Project Genesis is the internal Certiaura repository manager.

## Purpose

The application will eventually provide:

- Repository status dashboard
- Asset register viewer
- Build pack import
- Repository validation
- Commit and push controls
- Release creation
- Completeness scoring
- Asset relationship checks

## Current version

v1.0 scaffold — initial desktop application structure.

## Technology

Python + Tkinter for the first prototype.

This is intentionally simple so it can run locally on Windows without needing a web server.

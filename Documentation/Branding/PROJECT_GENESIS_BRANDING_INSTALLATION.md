# PROJECT GENESIS BRANDING INSTALLATION

After importing and committing Build 0014:

1. Close Project Genesis.
2. Open `13_Project_Genesis\app`.
3. Double-click `INSTALL_PROJECT_GENESIS_BRANDING.bat`.
4. Reopen `run_genesis.bat`.

The installer preserves the original launcher as `run_genesis.pre_branding_backup.bat`.

Use `RESTORE_PRE_BRANDING_LAUNCHER.bat` to restore the original launcher.

# PEPTIDE BATCH B — IDENTITY & DUPLICATION CONTROL

## GHK-Cu

Distinguish:

- free GHK;
- copper-complexed GHK-Cu;
- cosmetic copper tripeptide-1;
- injectable blends marketed as “Glow”.

## Thymosin Beta-4 / TB-500

Distinguish:

- full-length endogenous thymosin beta-4 / timbetasin;
- synthetic full-length thymosin beta-4;
- TB-500 / Ac-LKKTETQ fragment;
- supplier products labelled only “thymosin”.

## Melanotan II

Distinguish:

- Melanotan II;
- afamelanotide / Melanotan I;
- bremelanotide;
- unverified tanning sprays and injections.

## Epitalon

Distinguish:

- synthetic AEDG tetrapeptide;
- epithalamin pineal extract;
- modified analogues;
- blends marketed as longevity products.

# Build 0037 Source Reference Register

The package control framework was checked against current official UK sources including:

- CAP Code Section 12 — medicines, medical devices, health-related products and beauty products;
- MHRA Blue Guide — advertising and promotion of medicines in the UK;
- MHRA software and artificial intelligence as a medical device guidance;
- ICO guidance on artificial intelligence, data protection, automated decision-making and profiling;
- CMA guidance on unfair commercial practices under the Digital Markets, Competition and Consumers Act 2024.

The machine-readable authoritative links and review dates are maintained in:

`06_Evidence/Regulatory/UK_HEALTH_CLAIMS_ADVERTISING_AI_SOURCE_REGISTER.json`

This register supports operational control design. A live release decision must confirm the current source position and obtain specialist review where the claim, product, audience or intended purpose requires it.

# RETATRUTIDE REGULATORY MONITORING SPECIFICATION

## Monitoring mode

**ACTIVE CONDITION WATCH**

## Search priority

1. FDA
2. MHRA
3. EMA / European Commission
4. WADA
5. Eli Lilly official regulatory / investor communications
6. High-quality reporting only when primary authority access is unavailable

## Notify only for material change

Examples:

- new drug application or marketing-authorisation submission;
- application acceptance;
- priority review;
- advisory committee action;
- approval or rejection;
- safety communication;
- recall;
- enforcement;
- anti-doping status change;
- formal launch authorisation.

## Notification content

State:

- what changed;
- date;
- jurisdiction;
- authority;
- source;
- previous status;
- new status;
- affected CertiAura files;
- commercial and safety implications.

## Silence rule

If nothing material changed, do not notify.

# REGULATORY MONITORING AUTOMATION STANDARD

A regulatory watch should:
1. Check authoritative sources first.
2. Look for material change since the last verified record.
3. Notify only when something materially changes.
4. State what changed, date, authority, source, affected assets and implications.
5. Remain silent when nothing material changed.

Every peptide or compound must state one monitoring mode:
- ACTIVE
- EVENT-TRIGGERED
- MANUAL REVIEW
- NOT APPLICABLE

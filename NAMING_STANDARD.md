# Naming Standard

## Universal Asset Identifier

Format:

```text
CERT-[SYSTEM]-[NUMBER]
```

Examples:

```text
CERT-PKS-000001
CERT-BKS-000001
CERT-CKS-000001
CERT-MKS-000001
CERT-EKS-000001
CERT-GKS-000001
```

## File naming

Preferred format:

```text
ASSET-ID_Asset_Name_vX.Y.md
ASSET-ID_Asset_Name_vX.Y.json
```

Example:

```text
CERT-PKS-000001_Retatrutide_v1.1.md
CERT-PKS-000001_Retatrutide_v1.1.json
```

## Folder naming

Use numbered folders for top-level repository areas.

Use underscores instead of spaces.

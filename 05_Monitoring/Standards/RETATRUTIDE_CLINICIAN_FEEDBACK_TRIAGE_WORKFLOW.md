# Retatrutide Clinician Feedback Triage Workflow

Build provenance: CERT-BUILD-0049

## Permitted dispositions

- `NO_CHANGE`
- `CLARIFICATION_REQUIRED`
- `DATA_CORRECTION_REQUIRED`
- `CLINICAL_REVIEW_REQUIRED`
- `EXPORT_AMENDMENT_REQUIRED`
- `LOCKED_URGENT_ROUTING`

Feedback is classified, provenance-linked and retained as an audit record. It must not contain direct identifiers or autonomous treatment instructions. Where an amendment is required, the original approved export remains immutable and a new export version is generated through the Build 0048 approval controls.

# UNIVERSAL REGULATORY & STATUS INTELLIGENCE FRAMEWORK

## Scope
Applies to peptides, peptide analogues, small molecules, biologics, blends, investigational compounds and future CertiAura product-linked compounds.

## Mandatory domains
- Approval / authorisation
- Prescribing / dispensing
- Compounding where relevant
- Anti-doping where relevant
- Safety communications
- Recalls
- Enforcement
- Scheduled regulatory events
- Commercial implications

## Required fields
- Asset UAI
- Asset name
- Jurisdiction
- Authority
- Status category
- Current status
- Status date
- Source date
- Source URL or identifier
- Source type
- Source closure state
- Commercial implication
- User warning
- Next review trigger
- Next review date where known
- Monitoring state
- Last verified
- Change history

## Source hierarchy
1. Regulator / authority
2. Official government legal source
3. Official anti-doping authority
4. Official sponsor or trial-registry regulatory communication
5. High-quality reporting when primary access is temporarily unavailable
6. Secondary source only as a labelled lead

## Freshness states
- CURRENT
- EVENT PENDING
- REVIEW DUE
- STALE
- SUPERSEDED
- UNKNOWN / UNVERIFIED

## Material-change response
1. Capture source.
2. Verify effective date.
3. Compare prior status.
4. Update dated record.
5. Review claims.
6. Review commercial boundary.
7. Update Product Passport™ where relevant.
8. Record change.

## Commercial separation
Never conflate:
- studied;
- approved;
- legally prescribed;
- legally compounded;
- legally sold;
- sold online;
- WADA-permitted;
- product-quality verified.

# Retatrutide Case Closure Governance and Safety Boundary

Build provenance: `CERT-BUILD-0050`

This workflow is a controlled information-governance baseline. It does not diagnose, prescribe, select treatment, change dose, determine clinical suitability or replace a clinician.

Direct identifiers are prohibited. Urgent-routing precedence is immutable. Closure can be reopened when new evidence, feedback, safety information, data correction or an unresolved action emerges.

# Retatrutide Outcome Trend Visualisation Standard

## Requirements

1. Render locally without network dependencies.
2. Use deterministic SVG and a SHA-256-linked rendering manifest.
3. Use text content derived from bounded analytics fields only.
4. Show insufficient-data and urgent-routing states.
5. Include a visible descriptive-only safety boundary.
6. Do not embed scripts, remote URLs or persistent storage.

# Retatrutide Evidence-Linked Analytics Standard

## Requirements

1. Inputs must be pseudonymised and identifier-screened.
2. Metrics must be descriptive, deterministic and source-traceable.
3. Every metric must retain source event identifiers, source references and configured evidence references.
4. Fewer than the configured minimum observations must produce `INSUFFICIENT` data state.
5. Urgent routing locks must propagate without interpretation.
6. No output may diagnose or recommend treatment, dose, titration or procurement.
